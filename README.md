# React Tilt


A hover effect container written in React.js

Not mobile-friendly at the moment, planning on mobile implementation soon.

## Installation

`yarn add react-tilt`

or

`npm install react-tilt`



## Usage

```javascript
import Tilt from 'react-tilt';
	
export default () => {
		
	return (
		<Tilt>
			My Tilted content
		</Tilt>
	);
}
```
